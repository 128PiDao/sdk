## ⚙️ Installation

```bash
# Clone repository
git clone https://github.com/yourusername/128pi.git

# Masuk ke folder project
cd 128pi

# Install dependencies
npm install
```

---

## ▶️ Usage

```bash
# Run development
npm run dev

# Build project
npm run build

# Start production
npm start
```

## 🔧 Configuration

Buat file `.env` berdasarkan `.env.example`:

```env

API_KEY=your_api_key
NODE_ENV=development
```
